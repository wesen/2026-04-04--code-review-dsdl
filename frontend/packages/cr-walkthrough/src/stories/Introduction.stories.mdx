# CR Walkthrough — Component Library

The `@crs-cradle/cr-walkthrough` Storybook documents every component and step renderer in the CR Walkthrough System.

## Theming

All visual regions are controlled by CSS custom properties on `[data-widget="cr-walkthrough"]`. Override any token:

```tsx
<CRWalkthrough
  walkthrough={wt}
  tokens={{
    '--cr-color-source-step': '#ff6600',
    '--cr-radius-lg': '0px',
  }}
/>
```

## data-part attributes

Every visual region has a stable `data-part` attribute. Use these in CSS selectors instead of class names:

```css
[data-widget="cr-walkthrough"] [data-part="file-badge"] {
  background: var(--cr-color-surface-raised);
}
```

## Stories by section

- **Primitives** — `FileBadge`, `CodeBlock`, `CodeLine`, `Note`
- **StepCard** — all 13 step types in card form
- **Interactive** — `Checkpoint`, `Reveal`, `Branch` (stateful)
- **Annotation/Section** — all 4 severities + SourceRenderer
- **CRWalkthrough** — root component stories (dark, light, unstyled, custom tokens)
- **Introduction** — this page

## Story backgrounds

Stories are shown against both `#101114` (dark) and `#ffffff` (light) backgrounds. Use the Storybook ** backgrounds** panel to switch.
